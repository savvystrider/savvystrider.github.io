The basic data types in JSON are referred to as **values**.

### Strings
Text enclosed in double-quotation marks.

!!! example

    ```json
    "example"
    ```

### Numbers
- Integer (whole) or decimal
- Positive or negative
- Zero
- No quotation marks

!!! example

    ```json
    [1, 2.5, 0, 382]
    ```

### Booleans
- True or false
- No quotation marks

!!! example

    ```json
    [true, false, false, true]
    ```

### Null

- Means "nothing"
- No quotation marks

!!! example

    ```json
    [null]
    ```
